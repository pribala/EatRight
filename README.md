# Eat-Right!! Recipe search with React

###This app allows users to search the Edamam API for recipes according to keyword, calories, diet type. Users can save,comment favourite recipes and view them. 

## Technologies
* MongoDB
* Node Packages
* cheerio
* HTML
* CSS
* expresss
* JQuery
* Bootstrap
* React JS
